# Основы машинного обучения: Практические работы

В данном репозитории представлены учебные материалы и программные реализации базовых алгоритмов машинного обучения. Основной упор сделан на понимание математической сути алгоритмов и их реализацию «с нуля» на языке Python.

## Содержание

### [Практика 1: Предобработка данных](Практика%201%20(1).ipynb)
* Работа с библиотекой **Pandas**.
* Очистка данных и базовая обработка признаков.
* Реализация фильтра по низкой дисперсии (**Low Variance Filter**) для отбора информативных признаков.

### [Практика 2: Линейные дискриминантные функции](Практика%202%20(2).ipynb)
* Построение линейной дискриминантной функции $g(x) = \omega^T x + w_0$.
* Вычисление логитов и применение функции **Softmax** для получения вероятностей.
* Расчет информационной энтропии для оценки неопределенности предсказаний.

### [Практика 3: Геометрические методы классификации](Практика%203%20(2).ipynb)
* Аналитическое доказательство линейности гиперплоскости при использовании евклидова расстояния.
* Реализация классификатора на основе **центроидов** классов.
* Визуализация распределения классов и решающих границ.

### [Практика 4: Линейная регрессия и регуляризация](Практика%204%20(1).ipynb)
* Реализация парной линейной регрессии аналитическим методом.
* Расчет ковариации и коэффициентов регрессии $\beta_0, \beta_1$.
* Реализация шага градиентного спуска для **Elastic Net** (L1 + L2 регуляризация).

### [Практика 5: Метрики качества регрессии](Практика%205%20(2).ipynb)
* Самостоятельная реализация ключевых метрик оценки моделей:
  * MSE, RMSE, MAE.
  * Коэффициент детерминации ($R^2$).
  * MAPE, WAPE и BadFreq.

### [Практика 6: Многоклассовая классификация и отступы](Практика%206%20(2).ipynb)
* Расчет уравнения разделяющей прямой в пространстве $R^2$.
* Вычисление **Margin** (отступа) для объектов выборки для анализа качества разделения классов.

### [Практика 7: Стратегии ансамблирования (One-vs-One)](Практика%207%20(1).ipynb)
* Реализация стратегии **OvO** для решения задач многоклассовой классификации.
* Создание и обучение набора бинарных классификаторов для каждой пары классов.

### [Практика 8: Методы оптимизации](Практика%208%20(1).ipynb)
* Реализация алгоритмов минимизации функции потерь:
  * Классический **градиентный спуск**.
  * **Метод Ньютона** с использованием матрицы Гессе для ускорения сходимости.
